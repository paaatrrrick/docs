## Image

### Utilisation du Markdown

La [syntaxe markdown](https://www.markdownguide.org/basic-syntax/#images) vous permet d'ajouter des images en utilisant le code suivant

```md
![titre](/chemin/image.jpg)
```

Notez que la taille du fichier image doit être inférieure à 5 Mo. Sinon, nous recommandons l'hébergement sur un service comme [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Vous pouvez ensuite utiliser cette URL pour l'intégration.

### Utilisation des intégrations

Pour plus de personnalisation avec les images, vous pouvez également utiliser les [intégrations](https://mintlify.com/docs/text#embeds) pour ajouter des images

```html
<img height="200" src="/chemin/image.jpg" />
```

### iFrames

Charge une autre page HTML dans le document. Utilisé le plus souvent pour intégrer des vidéos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
