Patrick's personal website is a React-based portfolio that displays his professional experience and side projects.

## Tech stack

- **Frontend**: React 18 with React Router for navigation
- **Styling**: Custom CSS with animations and SVG wave dividers
- **Deployment**: Netlify with automatic redirects
- **Assets**: Cloudinary for image and video hosting
