# Demos

Explore live demonstrations and interactive examples to see Mintlify in action. These demos showcase key features and help you understand how to implement similar functionality in your own documentation.

## Live examples

Browse our collection of interactive demos to see Mintlify's features in real-world scenarios.

### Basic documentation

See how Mintlify renders standard documentation components:

- **Text formatting** - Headers, paragraphs, lists, and emphasis
- **Code blocks** - Syntax highlighting and copy functionality  
- **Images and media** - Responsive images and video embeds
- **Navigation** - Sidebar navigation and page linking

### Interactive components

Experience Mintlify's advanced interactive features:

- **API playground** - Test API endpoints directly in the documentation
- **Tabs and accordions** - Organize content with collapsible sections
- **Callouts and alerts** - Highlight important information
- **Code snippets** - Reusable code examples across multiple pages

### Customization showcase

Explore different themes and customization options:

- **Custom branding** - See how to apply your brand colors and logos
- **Dark mode** - Toggle between light and dark themes
- **Custom components** - Examples of MDX components and extensions
- **Mobile responsive** - Test documentation on different screen sizes

## Getting started with demos

To run these demos locally:

1. Clone the demo repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
4. Browse to `http://localhost:3000` to explore

## Contributing demos
