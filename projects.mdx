## Chemistry research

The Instructor Facilitation Coding Scheme visualization tool helps analyze chemistry teacher communication approaches. Available at [facilitation-visualization.netlify.app](https://facilitation-visualization.netlify.app/).

### Features

- **Excel file upload** - Process teaching session data from spreadsheets
- **Configurable visualization** - Adjust width, seconds per check mark, and check mark intervals
- **Interactive canvas** - Generate visual representations of facilitation coding data
- **Multiple shape types** - Represents different teaching behaviors with shapes:
  - Rectangles for task phases (introduction, during, closing)
  - Triangles for questioning behaviors
  - Circles for relaying information
  - Diamonds and hexagons for special interactions

### Coding categories

- **Managing** - Classroom management, directions, encouragement, announcements
- **Questioning** - Content questions, non-content questions, rhetorical questions
- **Relaying** - Answering questions, explaining answers, providing hints
- **Communicative approach** - Interactive/non-interactive, authoritative/dialogical
