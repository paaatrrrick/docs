## Chemistry research

An instructor facilitation coding scheme visualization tool for chemistry education research.

**Key features:**
- Excel file upload for data processing
- Canvas-based visualization with configurable dimensions
- Time-based graphical representation of instructor communication techniques
- Support for multiple shape types (rectangles, circles, triangles, diamonds, hexagons)
- Customizable width, seconds per check mark, and check mark intervals

**Communication approach categories:**
- Interactive/Non-interactive Authoritative
- Interactive/Non-interactive Dialogical
- Managing activities (announcements, directions, encouragement)
- Questioning types (content, non-content, rhetorical)
- Relaying information (answers, hints, analogies)
