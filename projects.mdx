## Site personnel

Le site personnel de Patrick à [patrickrfoster.com](https://www.patrickrfoster.com) est un portfolio basé sur React présentant ses expériences et projets.

### Stack technique

- React 18 avec React Router
- Stylisation CSS avec animations personnalisées
- Déployé via Netlify

### Projets en vedette

- **Courser** - ChatGPT pour les salles de classe universitaires, gagnant de HackUIowa 2023
- **Assistant d'écriture IA (Wordsmith)** - Extension Chrome utilisant GPT pour réécrire le texte surligné
- **World's Best Boilerplate** - Starter d'application web full-stack avec authentification, barre de navigation, gestion des erreurs et base de données
- **Télécommande à gestes** - Contrôle de téléviseur basé sur OpenCV et régression linéaire
- **Trouveur de matchs de tennis (Actively)** - Application de rencontres sportives pour trouver des matchs à proximité

### Fichiers clés

- `src/constants/projects.js` - Définitions des données de projet
- `src/Home.js` - Composant principal de la page d'accueil

## Recherche en chimie

Le [Schéma de codage de la facilitation de l'instructeur](https://facilitation-visualization.netlify.app/) est un outil de visualisation pour la recherche en éducation chimique.

### Objectif

Cet outil permet aux chercheurs de :

- Télécharger des fichiers Excel avec des données de facilitation de l'instructeur
- Générer des représentations visuelles des techniques de communication
- Configurer la largeur de sortie, les intervalles de temps et la fréquence des coches
- Exporter des images haute résolution pour l'analyse

### Stack technique

- JavaScript vanilla avec Fabric.js pour le rendu canvas
- Bibliothèque read-excel-file pour l'analyse Excel
- Stylisation CSS personnalisée

### Fichiers clés

- `app.js` - Logique principale de l'application avec rendu canvas Fabric.js
- `index.html` - Interface de téléchargement et options de configuration
