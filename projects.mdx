## Chemistry research

The [Instructor Facilitation Coding Scheme](https://facilitation-visualization.netlify.app/) is a visualization tool for chemistry education research.

### Purpose

This tool allows researchers to:

- Upload Excel files with instructor facilitation data
- Generate visual representations of communication techniques
- Configure output width, time intervals, and check mark frequency
- Export high-resolution images for analysis

### Tech stack

- Vanilla JavaScript with Fabric.js for canvas rendering
- read-excel-file library for Excel parsing
- Custom CSS styling

### Key files

- `app.js` - Main application logic with Fabric.js canvas rendering
- `index.html` - Upload interface and configuration options
