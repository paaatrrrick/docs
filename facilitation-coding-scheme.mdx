The [Instructor Facilitation Coding Scheme](https://facilitation-visualization.netlify.app/) is a web-based tool that converts Excel data into visual timelines for analyzing classroom instructor facilitation behaviors.

## Features

- Upload Excel files containing coded facilitation data
- Generate visual timelines with customizable width and time intervals
- Export high-resolution images for research and presentations

## How to use

1. Navigate to the [visualization tool](https://facilitation-visualization.netlify.app/)
2. Configure your settings:
   - **Width**: Set the output image width (default: 4000px)
   - **Seconds per check mark**: Time interval for timeline markers (default: 60 seconds)
   - **Check marks until labeled**: Interval for labeled markers (default: 10)
3. Upload your Excel file with coded data
4. Double-click the generated image and select "Save Image as..." to export

## Coding categories

The tool supports visualization of several facilitation categories:

### Task phases

| Category | Description |
|----------|-------------|
| Introduction of Task | Opening phase of an activity |
| During Task | Active working phase |
| Closing Task | Concluding phase of an activity |

### Communicative approaches

- Interactive Authoritative
- Noninteractive Authoritative
- Interactive Dialogical
- Noninteractive Dialogical

### Managing behaviors

Includes classroom management actions like:
- Announcing/closing question periods
- Calling on students
- Giving directions
- Course reminders
- Encouraging collaboration
- Time information

### Questioning behaviors

Tracks various question types:
- Content questions (answered/unanswered)
- Non-content questions
- Whole class responses
- Cold calls
- Rhetorical questions

### Relaying behaviors

Captures instructor responses:
- Answering student questions
- Explaining answers
- Giving hints and analogies
- Providing task answers
- Responding to student answers

### Interaction types

- Instructor initiated group interaction
- Instructor initiated whole class interruption
- Student initiated group interaction
- Student initiated whole class interruption
